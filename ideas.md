# Brainstorm de Design - Portfólio Eduardo Pina

## Contexto
Portfólio profissional para Eduardo Pina Fleury Fortuna, Desenvolvedor Backend com foco em Java, Node.js e ERP Sankhya. Objetivo: reforçar autoridade técnica, posicionar como profissional em evolução para pleno, destacar experiência com sistemas corporativos legados.

---

## Ideia 1: Minimalismo Corporativo com Destaques Tech

**Design Movement:** Modernismo corporativo com influências de design system (similar a Stripe, GitHub)

**Core Principles:**
- Tipografia clara e hierarquizada como protagonista
- Espaçamento generoso e respiração visual
- Austeridade com destaques estratégicos em azul/verde tech
- Foco em conteúdo, não em decoração

**Color Philosophy:**
- Background: Branco puro ou cinza muito claro (quase branco)
- Foreground: Cinza escuro/preto para texto
- Accent: Azul profundo (#0066CC ou similar) para CTAs e destaques
- Secundário: Verde tech suave (#00AA66) para badges de skills
- Reasoning: Transmite profissionalismo, confiança, clareza técnica

**Layout Paradigm:**
- Seções com assimetria controlada
- Alternância entre full-width e constrained layouts
- Grid de 2-3 colunas para cards de experiência
- Uso de linhas divisórias subtis em vez de cores

**Signature Elements:**
- Cards com border-left colorido (azul/verde) em vez de sombra
- Badges de skills com background suave e texto bold
- Ícones minimalistas (Lucide) em azul como acentos
- Linhas horizontais finas como separadores

**Interaction Philosophy:**
- Hover: Mudança sutil de cor de fundo (cinza muito claro)
- Transições suaves (0.3s) em todas as interações
- Underline animado em links
- Nenhuma animação excessiva, tudo muito controlado

**Animation:**
- Fade-in ao scroll (opacity 0→1, 0.6s ease-out)
- Slide-up suave para cards (transform translateY, 0.5s)
- Hover em botões: background color shift + scale(1.02)
- Nenhuma rotação ou efeito 3D

**Typography System:**
- Display: Poppins Bold (24-32px) para headings principais
- Body: Inter Regular (16px) para texto corrido
- Mono: Fira Code (12-14px) para snippets de código
- Hierarchy: Peso e tamanho, não cor

## Ideia 2: Dark Mode Elegante com Acentos Vibrantes

**Design Movement:** Cyberpunk sofisticado (influência de Vercel, Framer)

**Core Principles:**
- Dark mode como padrão, transmitindo modernidade e sofisticação
- Acentos vibrantes em verde neon ou azul elétrico
- Gradientes sutis como elementos de profundidade
- Movimento e interatividade como parte central da experiência

**Color Philosophy:**
- Background: Preto profundo (#0A0A0A) ou cinza muito escuro (#1A1A1A)
- Foreground: Branco/cinza claro para texto
- Accent: Verde neon (#00FF88) ou azul elétrico (#00D9FF)
- Secundário: Roxo suave (#7C3AED) para destaques secundários
- Reasoning: Moderno, tech-forward, atrai atenção de recrutadores em portfólios

**Layout Paradigm:**
- Seções com fundo gradiente sutil (preto → cinza escuro)
- Uso de glassmorphism (cards com backdrop-blur)
- Diagonais e ângulos assimétricos em dividers
- Sobreposição de elementos com diferentes profundidades

**Signature Elements:**
- Glow effect em torno de elementos interativos (azul/verde)
- Animated gradient borders em cards
- Ícones com efeito de brilho (glow)
- Linha animada como separador entre seções

**Interaction Philosophy:**
- Hover: Glow intenso + mudança de cor
- Clique: Pulse animation (escala + opacidade)
- Scroll: Parallax suave em backgrounds
- Tudo com transições fluidas e satisfatórias

**Animation:**
- Glow pulse contínuo em elementos destacados
- Slide-in com rotate suave para cards (rotateY 10deg)
- Gradient animation em backgrounds (20s loop)
- Hover: Scale + glow intenso (0.3s cubic-bezier)
- Parallax em hero section

**Typography System:**
- Display: Space Mono Bold (28-36px) para headings (monospace tech feel)
- Body: Inter Regular (16px) para texto corrido
- Accent: Fira Code para código e skills técnicos
- Hierarchy: Peso, tamanho e cor (accent color para destaque)

## Ideia 3: Minimalismo Japonês com Destaques Azuis

**Design Movement:** Wabi-sabi digital (simplicidade, espaço negativo, naturalidade)

**Core Principles:**
- Máxima simplicidade com máximo impacto
- Espaço negativo como elemento de design
- Tipografia como único elemento decorativo
- Paleta reduzida (3-4 cores apenas)

**Color Philosophy:**
- Background: Branco ou off-white (#FAFAFA)
- Foreground: Cinza muito escuro (#2A2A2A)
- Accent: Azul profundo e sereno (#1E40AF)
- Secundário: Cinza suave (#D1D5DB) para borders
- Reasoning: Elegância através da restrição, foco total no conteúdo

**Layout Paradigm:**
- Máximo espaçamento entre elementos
- Seções com breathing room generoso
- Alinhamento rigoroso à esquerda ou centro
- Nenhuma decoração, apenas conteúdo e espaço

**Signature Elements:**
- Linha vertical fina (1px) como divisor entre seções
- Números grandes em azul como destaque de métricas
- Citações em itálico como elemento visual
- Ícones muito pequenos e sutis

**Interaction Philosophy:**
- Hover: Apenas mudança de cor de texto (para azul)
- Nenhum efeito de sombra ou escala
- Transições muito sutis (0.2s)
- Foco em clareza, não em wow factor

**Animation:**
- Fade-in ao scroll (muito suave, 1s ease-in-out)
- Nenhuma animação contínua
- Hover: Color change apenas (0.2s)
- Nenhuma transformação de escala ou rotação

**Typography System:**
- Display: Playfair Display Bold (32-48px) para headings (elegância clássica)
- Body: Lora Regular (16px) para texto (serif elegante)
- Mono: Courier New (12px) para código
- Hierarchy: Tamanho e espaçamento, não peso

---

## Seleção Final

**Abordagem Escolhida: Ideia 2 - Dark Mode Elegante com Acentos Vibrantes**

**Justificativa:**
- Alinha perfeitamente com a referência visual (Guga Silva - dark mode, animações, modernidade)
- Transmite profissionalismo tech-forward, atraindo recrutadores de empresas modernas
- Permite destacar habilidades técnicas com acentos vibrantes
- Animações suaves e interatividade reforçam a experiência de um dev moderno
- Verde neon (#00FF88) ou azul elétrico (#00D9FF) como accent cria visual memorável
- Dark mode é tendência em portfólios de devs (GitHub, Vercel, etc.)

**Paleta Final:**
- Background: #0A0A0A (preto profundo)
- Card Background: #1A1A1A (cinza escuro)
- Text: #F5F5F5 (branco suave)
- Accent Primary: #00D9FF (azul elétrico)
- Accent Secondary: #7C3AED (roxo suave)
- Border: #333333 (cinza escuro com contraste)

**Fontes:**
- Display: Space Mono Bold (Google Fonts)
- Body: Inter Regular (Google Fonts)
- Code: Fira Code (Google Fonts)

**Animações Principais:**
- Fade-in ao scroll (0.6s ease-out)
- Glow effect em hover (cards e botões)
- Slide-in com rotate suave (cards)
- Gradient animation em backgrounds (20s loop)
- Parallax suave em hero section
